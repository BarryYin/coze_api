# coze_api_cookbook
结合coze_api的独立页面

### 项目起源
coze.cn 的api特别难用，因此写了此野生cook-book

### 使用指南
- 本项目适配coze_bot 的api
- 运行本项目只需修改bot_id, 以及身份鉴权的token

### 安装环境

安装python

### 运行步骤
- 启动文件test_bot.py

python test_bot.py

- 等待启动成功后，会出现网页地址，点击即可

 * Running on http://127.0.0.1:5000
INFO:werkzeug:Press CTRL+C to quit
